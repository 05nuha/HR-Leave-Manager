HR Leave Manager

A complete leave management solution for HR teams — built for Odoo 19

Overview

HR Leave Manager gives your organisation full control over employee leave — from initial request to final approval. It includes a real-time dashboard, multi-view analytics, automated workflows, and printable leave documents.

Key Features

Live Dashboard

At-a-glance overview of all leave activity — draft, pending, approved, and refused — with clickable stat cards and percentage breakdowns.

Leave Requests

Full approval workflow with Submit, Approve, and Refuse actions. Tracks state changes via the chatter with automatic email notifications.

Allocations

Manage how many days each employee is entitled to per leave type, with remaining days calculated in real time.

Leave Types

Configure leave types with colour coding, max days, and approval requirements. Annual, Sick, Emergency, Maternity, and Study leave included out of the box.

Calendar View

Visualise all leave requests on a monthly calendar, colour-coded by leave type, for easy scheduling and conflict detection.

Graph & Pivot Analysis

Bar chart and pivot table views let managers analyse leave trends across employees, leave types, and approval states.

Overlap Validation

Automatically prevents employees from submitting overlapping leave requests for the same period.

PDF Reports

Print professional leave request documents with employee details, leave period, reason, and signature fields — ready for HR filing.

Automated Cron Job

A daily scheduled action automatically refuses any leave requests whose end date has passed and are still pending.
